# PortBay Homebrew Tap

Homebrew tap for [PortBay](https://portbay.app).

```sh
brew install --cask portbay-app/portbay/portbay
```

The `Casks/portbay.rb` formula is updated automatically by the
[release workflow](https://github.com/portbay-app/portbay/blob/main/.github/workflows/release.yml)
on each tagged release.
